# Mythos — AI Story Creator

An AI-powered web app that generates cinematic stories using OpenRouter.

## Features
- Scene-based storytelling
- Custom genre, hero, setting
- AI-generated content

## Setup

1. Download project
2. Run:
   npm install
   npm start

3. Create `.env` file:
   OPENROUTER_API_KEY=your_key

## Note
API key is required to run the project.
